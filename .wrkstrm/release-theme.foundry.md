# Release Theme (Foundry)

This file defines release codenames for swift-tradier-lib. The theme leans on
market data, streaming, and quote delivery.

## Naming format

- Canonical form: `<version> - <Codename>`
- Codename format: Title Case, two words preferred.
- Vocabulary: quotes, ticks, tape, sessions, signals.

## Major releases

- 1.0.0 - Market Current
- 2.0.0 - Signal Backbone

- 0.2.0 - Option Chain

- 0.9.0 - Sandbox Feed
- 0.10.0 - Live Wire
